# Brettspiel Klimafitte Stadtsanierung

Das Brettspiel "Klimafitte Stadtsanierung" zielt auf die spielerische Vermittlung des realen Spannungsfelds zwischen Kosten, technischer Machbarkeit und CO2 Reduktion in der Gebäudesanierung in einem rundenbasierten Strategiespie ab. 
Die Spieler schlüpfen dabei in die Rolle von Bauträgern und Planenden, die versuchen, durch geschicktes Erforschung von Technologien und Umsetzen von ökologischen Sanierungsprojekten möglichst viel CO2 einzusparen. 

*Zielgruppe*: Das Spiel richtet sich an interessierte Jugendliche und Erwachsene (16-99J)
